# 🔴 Islam Facing the Red Tide — الإسلام في وجه الزحف الأحمر

**بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ**

Interactive trilingual (AR/EN/FR) educational web app based on the book **الإسلام في وجه الزحف الأحمر (Islam Facing the Red Tide)** by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- 20 confrontation cards with Quranic verses, 5 Islam vs. communism contrasts, daily habit tracker, knowledge quiz
- Trilingual: Arabic, English, French with RTL/LTR support
- 3 themes, narrator, responsive design, PWA-ready

## Tech

Pure HTML + CSS + JS. No frameworks. Just open `index.html`.
